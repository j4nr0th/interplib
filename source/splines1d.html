1D Splines#

Splines are functions which are defined piecewise on an interval. There are two main types of splines provided in interplib:

  • Integer splines, implemented in Spline1Di,

  • Regular splines, implemented in Spline1D.

Note that these use same power series representation as Polynomial1D and as such have the same weaknesses and strengths.

To construct splines based on either nodal values or element averages, convenience functions are provided. All of them use use of SplineBC type to define the boundary conditions.

class interplib.SplineBC(coeffs: array_like, value: float)[source]#

Represents a boundary condition for a spline.

This represents the boundary condition expressed as:

\(\sum\limits_{n=0}^N k_n \frac{d^n}{{dt}^n}(y) = v\),

where the values \(k_n\) correspond to entries in coeffs[n].

Parameters:

  • coeffs ((N,) array_like) – Represents coefficients of the spline’s derivatives.

  • value (float) – Value of the weighted sum for the boundary condition.

class interplib.Spline1D(nodes: array_like, coefficients: array_like, /)#

Piecewise polynomial function, defined between nodes. Given the set of nodes

\(\left\{x_0, \dots, x_n \right\}\) and sets of coefficients \(\left\{A_0, \dots A_{n-1} \right\}\), the polynomial evaluated between nodes \(x_i\) and \(x_{i + 1}\) will be:

\[p(x) = \sum\limits_{k=0}^M A_i^k \left( 2 \frac{x - x_i}{x_{i+1} - x_{i}} - 1 \right)^k,\]

where \(A_i^k\) is the k-th coefficient of the set \(A_i\).

These splines thus allow for simple stitching of solutions of different 1D elements together, as those are typically defined on a reference element, where the computational space \(\xi \in \left[-1, +1\right]\) is then mapped to physical space.

Parameters:

  • nodes ((N + 1,) array_like) – Sequence of \(N + 1\) values, which mark where a different set of coefficients is to be used.

  • coefficients ((N, M) array_like) – Sequence of \(M\) coefficients for each of the \(N\) elements.

nodal_interpolating_splinei(n: int, nds: array_like, bcs_left: Iterable[SplineBC] | None = None, bcs_right: Iterable[SplineBC] | None = None) → Spline1Di#

Create interpolating spline, which has specified nodal values.

Parameters:

  • n (int) – Order of the spline. Must be odd, otherwise a ValueError will be raised.

  • nds ((N,) array_like) – Nodes to hit.

  • bcs_left (Iterable of SplineBC, optional) – Boundary conditions of the left side of the spline (must be of order n). If it is not provided, the natural boundary conditions are used.

  • bcs_right (Iterable of SplineBC, optional) – Boundary conditions of the right side of the spline (must be of order n). If it is not provided, the natural boundary conditions are used.

Returns:

Interpolating spline which maps from computational space \([0, N-1]\) to values.

Return type:

Spline1Di

element_interpolating_splinei(n: int, avgs: array_like, bcs_left: Iterable[SplineBC] | None = None, bcs_right: Iterable[SplineBC] | None = None) → Spline1Di#

Create interpolating spline, which has specified averages.

Parameters:

  • n (int) – Order of the spline. Must be even, otherwise a ValueError will be raised.

  • avgs ((N,) array_like) – Averages of elements to hit.

  • bcs_left (Iterable of SplineBC, optional) – Boundary conditions of the left side of the spline (must be of order n). If it is not provided, the natural boundary conditions are used.

  • bcs_right (Iterable of SplineBC, optional) – Boundary conditions of the right side of the spline (must be of order n). If it is not provided, the natural boundary conditions are used.

Returns:

Interpolating spline which maps from computational space \([0, N-1]\) to values.
